A full-stack doctor appointment booking application built with React.js, Node.js, Express.js, MongoDB, and Stripe. 
This project allows patients to book appointments, doctors to manage schedules, and administrators
to oversee the platform through a secure and responsive interface.

Features
User authentication
Book and manage appointments
Doctor dashboard
Admin dashboard
Online payments with Stripe
Responsive UI
MongoDB database integration
Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Payment Gateway: Stripe
